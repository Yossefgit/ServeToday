Serve Today is an Android application designed to bridge the gap between volunteers in local communities and non-profits. Our project features event discovery, simplified registration, and tools for tracking and exporting volunteer hours.
Created by Yossef Eini and Ramzi Carter

Pre req

Android Studio
Github

How to run:
copy repository link - https://github.com/Yossefgit/ServeToday.git
Open android studio
clone the project in Android studio
Create an emulator via the device manager
click run

Features:
search for events using specific dates and locations via zip code
Filter volunteering oppertunities by things such as time or name
register to specific shifts you are interested in
have a profile which shows things such as hours volunteered
